Affinity 用于教育

Affinity 应用程序日趋成为教育工作者的正确选择—价格合理、快速、可靠，且功能丰富。在其令人惊叹的性能、精确度和专业工具中，最为显著的是 Affinity 应用程序为教育工作者和学生的利益提供了独特的组合。

从 2017 年开始提供简单、实惠且灵活的许可

从 2017 年年初开始，我们将提供 macOS 和 Windows 版的 Affinity Designer 和 Affinity Photo，同时将为教育机构提供全新的优惠许可。

  • macOS 和 Windows 混合许可证
  • 可以选择“按座席”或“按学生”进行许可
  • 许可两种应用程序时的附加折扣
