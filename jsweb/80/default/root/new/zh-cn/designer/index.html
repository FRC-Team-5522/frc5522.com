重新定义了设计工具

您所需要的专业矢量设计应用程序中的所有工具，从精确性令人惊叹的钢笔工具到超平滑的渐变工具，所有均经过细致考虑和精心开发，其工作方式—完全按照您希望的样子。

为您的工作流程构建

基于性能卓越、稳定性超强和绝不复杂臃肿的核心原则，Affinity Designer 已针对专业工作流程进行精心制作。

跨平台性能

我们在 Mac 上铸造专业产品，并在 Windows 上逐步匹配，常见的文件格式在任一平台上都能提供令人惊叹的结果。

惊人的速度和准确性

以 60 英尺/秒的速度平移，缩放到惊人的 10,000,000% 倍，并能实时看到所有调整、效果、转换和画笔。

可应用任何颜色空间

RGB、CMYK、LAB、灰度。带有 ICC 颜色管理的端到端 CMYK 工作流程，以及每通道 16 位编辑。

无与伦比的文件兼容性

可用的最佳 PSD 导入引擎，并且全面支持 SVG、EPS、PDF、PDF/X 和 FH 文件，这意味着将 Affinity Designer 引入到您的工作流程中是毫不费力的。

像素的完美控制

只需点击一下就能以可用的标准或视网膜分辨率进行实时像素预览，您始终能够看到自己将会获得什么效果。无法以纯矢量达到完美的效果？只需编辑像素 - Affinity Designer 也有完整的一套栅格工具。

无损效果和调整

凭借巨大的调整图层、效果及混合模式库—并结合对蒙版和裁剪图层的完全支持—Affinity Designer 提供了任何基于矢量的应用程序中可用的最先进的图层控件。

灵活的工作空间… 和工作流程

Affinity Designer 采用集中、完全可定制的工作空间，包括停靠和浮动 UI 模式，可以让您按照自己想要的方式工作。它添加了常规和自定义 2D 以及等距网格可实现灵活的布局，能同时处理多个设计，并且它还使用 macOS 功能，如分屏显示和全屏。此外，通过可保存的历史记录、无限次撤消以及无损滤镜和调整，您可以随时后退并改变您的想法。

自然的笔法

采用压力触控、手写笔压力、倾斜和其他控件实现效果自然的艺术作品。编辑了矢量画笔笔划使其干净利落且流畅如同规则的曲线，并且还添加了栅格笔法—凭借其自身或与矢量图一同—增加了深度和高品质有机纹理。

旋转画布、混合颜色、编辑画笔参数、创建您自己的画笔和喷嘴，并导入 .abr 画笔以便完全掌控您的工作。

轻松给出某些形状

Affinity Designer 提供一套全面的形状图元，以及用于调整几何形状的边角设置和智能对齐指标等方便易用的控件。它还提供了全套布尔几何运算、无损复合形状，和节点完全编辑功能，这意味着您可以立即创造出美丽复杂的几何形状。

就是您所需的类型

添加了标题艺术文本、任何形状的文本框，或跟随任何矢量路径的文本。应用了复杂的样式和连字，可在屏幕上的面板中预览您的所有可用字体和样式设置。所有您所需要的控件均是内置的，包括行距、字距、基线偏移、制表位 – 而且现在在版本 1.5 中您可以创建跨文档的文本样式。

新的专业打印控件

Affinity Designer 具有全面的专业打印输出。Pantone® 支持、端到端 CMYK 和 ICC 颜色管理只是其中一部分。您还可以打开、编辑和输出 PDF/X 文件，设置套印控件，使用专色，以及添加出血区域、修剪和裁切标记。

强大的导出

在导出模式中，您将有一个专用于图像导出的完整工作空间。可以选择要导出的区域、图层或对象，并可独立控制各个项目的输出设置。为了实现高效的 Web 和 UI 设计，您可以在每当设计更改时以您希望使用的任何分辨率、文件格式和文件夹结构自动创建新文件。

常见的 Affinity 文件格式

Affinity 的共享文件格式使得混合规则设计呈现出其应有的流畅和简洁。在任何平台的任何 Affinity 应用中打开任何原生 Affinity 文件并继续工作 – 可共享历史记录、无限制撤销和无缝切换。对于导出，Affinity Designer 将所有主要文件类型都收入囊中。

极其强大的画板

在 Affinity Designer 中，您可以创建不限数量的不同设计变体、尺寸和画面 – 将它们放置在一个文档中以便同时查看所有方案。当您进行到导出时，您将能够完全控制输出哪些画板以及按照哪些设置输出，极大简化了您的工作流程的响应和应用程序设计。

我们有您的支持

虽然有经验的设计师会感到似曾相识，但我们有丰富的免费资源为您提供支持。这些资源包括几十个浅显易懂的在线视频教程、活跃的论坛和社交媒体反馈，您可以经常与某个开发人员直接交谈。 您也可以购买Affinity Designer 工作手册，通过这本令人惊艳的指南获取技术中的精华。

符号

将同一对象的多个实例链接到一起，编辑一个实例将同时更改所有其他实例。

限制

控制对象相对于其容器的位置和大小，在响应式样式环境中进行设计。

“资产”面板

在简单的拖放操作面板中整理所有资产，其中包含一个简单的方法来构建全面的画廊，并以您自己的文件夹结构进行组织。

实时嵌入式文档编辑

在当前设计中嵌入其他 Affinity 文件，以及在编辑嵌入式文档时实时查看结果。

路径上的文本

沿着自由绘制的直线、曲线和形状轮廓的流动文本，可以完全控制起点和基线。

大纲、像素和分屏显示视图

拆分屏幕显示，以普通视图、大纲模式，和以标准及视网膜分辨率的实时像素视图并排查看您的作品效果图。

边角工具

可准确围绕形状和绘制直线上的单个或多个选定边角，并且具有完美的几何形状。

即时撤消历史记录

通过使用“历史记录”滑块在数以千计的撤消步骤内立即取消。此外以文档保存您的历史记录以便从任何会话中撤消编辑步骤。

矢量画笔

绘制可编辑的笔划，将延伸或重复纹理画笔应用到矢量曲线，以及进行压力调整。我们已为您提供了一个大型库，您还可以轻松创建自己的库。

完全可定制的快捷键

凭借巨大的工具、菜单功能、视图和控件的键盘快捷键库来提高工作效率—您还可以选择自己的快捷键。
