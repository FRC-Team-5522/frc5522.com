经过数年的激情创造和对完美的不懈追求，Affinity 已经重新定义了照片的编辑和创意设计工作。

Affinity 提供的正是创意专业人士所需要的 – 飞快的速度、先进的工具、高精确度和无缝兼容性、以及富有吸引力的时尚的外观。

无论单独还是组合使用，快速、强大的处理过程都能很好地融合到您的工作流程内。单一文件格式使得可以在不同流派之间轻松转换，同时可保存的共享历史记录意味着您始终处理最新版本，不论您使用什么 Affinity 产品。

Affinity Designer

Affinity Designer 在 2015 年光荣地获得了“苹果设计奖”，在一款功能强大、优美的工具中为插图设计带来灵感。将矢量设计与栅格整理自然地整合到一个工具中，这个充满创意的插图工具将让您文思泉涌。简单地说，它的速度、功能和精度使您再次感受到艺术创作的乐趣和自发性，在 macOS 和 Windows 下都能获得高端成果。

Affinity Photo

荣获 2015 年度的苹果“年度应用”奖和科技影像记者协会颁发的 2016 年的“最佳图像处理软件”，Affinity Photo 是一款专业的照片编辑工具，拥有您所需要的全部功能。它的人像修饰工具实时运行，能够流畅即时地看到所做修改。Affinity Photo 的无损编辑、原始数据处理以及全程色彩管理是标准配置，因而能够提供生动地展现美丽的照片所需的所有质量。

Affinity Publisher

我们将在 2017 年推出 Beta 版本的 Affinity Publisher，届时将邀请您来帮助我们完善这个应用。毋庸置言，它也将具有强大的功能、速度、精确度和专业性，这是所有 Affinity 产品的核心，它将壮大我们的创意工具产品范围。

Affinity Publisher 旨在帮助专业设计人员在每一版面、页面、杂志、书籍和数字出版物中实现最佳的效果，展现令人惊艳的排版和绚丽的色彩。

未来

我们的 Affinity Photo 和 Designer iPad 版本在苹果全球开发者大会和大会以外的提前亮相引起了巨大的轰动。这两款产品将在 2017 年推出。

继对 Affinity Designer Windows 版的巨大需求之后，我们也将推出 Affinity Photo Windows 版。此外，一款数字资产管理工具也在计划之中。

我们的规模和结构意味着我们是快速而敏捷的，能够对变化的技术迅速做出反应，并且由于专门的团队针对不同的应用和平台，所以都获得同等的关注和重视。这意味着，无论您选择哪个版本的 Affinity 来满足您的需求，都能获得最高的质量。
